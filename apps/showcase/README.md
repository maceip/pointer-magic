# Magic Pointer local showcase

This is the local browser research surface for Magic Pointer.

- `/` — five working experiments: Threadline, Pattern Brush, Timefold, Source
  X-Ray, and Afterimage.
- `/after-chat` — the earlier Intent Halo, Shadow Run, and Apprentice Relay study.

## Run

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

The server binds only to the loopback interface. There is no authentication,
OpenAI Sites project, Cloudflare worker, hosted database, or external deployment
step.

## Other commands

```bash
npm run build
npm run lint
npm test
```
